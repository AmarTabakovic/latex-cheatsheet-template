# latex-cheatsheet-template
My personal template to typeset cheatsheets.

## Required Software
- pdflatex
- BibTeX
- GNU Make

Otherwise just modify the Makefile to your preferences.

## Building
### Compilation
```shell
$ make
```

### Removal of Auxilliary Files
```shell
$ make clean
```
